Scope

This document describes the planned releases of OpenLoad. No dates are given, the intention is to state which features will be available in which release, NOT when.

Releases

These are the planned releases:

  • 0.1.2 Better error handling (unable to find host, unable to connect to webserver)
  • 0.1.3 Bug fixes, config file / command line options for specifying request headers
  • 0.2.0 Move core measurement functions into a shared library, allowing several UI's to be build on top (Commandline, KDE, MFC, ...)
  • 0.2.x Added features: POST data, SSL?, ...
